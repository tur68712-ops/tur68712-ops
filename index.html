Lab 4: Choropleth Mapping Homicide Densities Map

This map was made to help visualize the spatial distribution of homicide densities across police districts in Chicago. The problem being solved by this map was how individuals could compare homicide activity between districts without needing to look at raw counts for each district.

To solve this, homicide data was summarized by district and then it was converted into a density measurement data using the calculate geometry and calculate field tools. A choropleth map was then made to help viewers visualize these values more easily. By looking at the map individuals can easily determine which districts have the highest/lowest homicide densities compared to the others.

GIS Tools and Skills Used

  • Table joins
  • Field creation
  • Calculation of field, geometry and density
  • Choropleth mapping techniques
  • Data analysis and export to conduct calculations
  • Map layout design
  • Project tool to correct coordinate systems
  • Layout tools
