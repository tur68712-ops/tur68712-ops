Throughout this semester, I was able to learn how to use GIS as a beginner to analyze real-world problems. Through this course I was able to learn how to communicate geographic patterns through maps and the use of GIS tools. This final lab report serves as my portfolio which highlights 3 of my strongest maps and the skills I developed in areas like spatial analysis, cartographic design, problem solving and data analysis in order to make the maps.

Lab 3: Proportional symbol Hispanic Population Map

This map was made to help visualize the distribution of the Hispanic Population across the contingent states. The main problem this map addressed was how someone could effectively compare population sizes between states without needing to look through massive amounts of separate data, on a clear and readable map. To achieve this I used proportional symbols to represent the hispanic population against a beige background to help improve readability. By looking at the map individuals interested in learning more about the Hispanic population distribution can easily determine that states such as California, Texas and New york have significantly higher Hispanic populations compared to other states.

GIS Tools and Skills Used

  • Proportional Symbol Mapping tools
  • Data Visualization skills
  • Visual Hierarchy Principles in aspects of color contrast and labeling
  • Map Layout Design to improve readability
  • Adjustment of symbology to ensure that the map was easy to read
  • Labeling tools to add abbreviated state names
